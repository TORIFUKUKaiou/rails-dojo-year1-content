## 問題1：自分だけの名刺

以下のような出力を作ってみましょう。罫線や記号は自由に変えてOKです。

```
+--------------------+
|  山田太郎          |
|  福岡県出身        |
|  好きな言語：Ruby  |
+--------------------+
```

<details>
<summary>解答例</summary>

```ruby
puts "+--------------------+"
puts "|  山田太郎          |"
puts "|  福岡県出身        |"
puts "|  好きな言語：Ruby  |"
puts "+--------------------+"
```

</details>

## 問題2：時間の計算

1日は何秒か、`puts` と `#{}` で計算して表示してみましょう。

```
1日は86400秒です
```

<details>
<summary>解答例</summary>

```ruby
puts "1日は #{24 * 60 * 60} 秒です"
```

</details>

## 問題3：単位変換

身長170cmをメートルに変換して表示してみましょう。

```
170cmは1mと70cmです
```

ヒント：`/` は割り算、`%` は余りを求める演算です。

```ruby
puts 170 / 100   # => 1
puts 170 % 100   # => 70
```

<details>
<summary>解答例</summary>

```ruby
puts "170cmは #{170 / 100}mと #{170 % 100}cmです"
```

</details>

## 問題4：九九の表（1段分）

好きな段を選んで、九九を横一列に表示してみましょう。

```
3の段：3 6 9 12 15 18 21 24 27
```

ヒント：`print` を使うと改行なしで表示できます。

```ruby
print "hello "
print "world"
# => hello world（1行に表示される）
```

<details>
<summary>解答例</summary>

```ruby
print "3の段："
print "#{3 * 1} "
print "#{3 * 2} "
print "#{3 * 3} "
print "#{3 * 4} "
print "#{3 * 5} "
print "#{3 * 6} "
print "#{3 * 7} "
print "#{3 * 8} "
print "#{3 * 9} "
puts ""
```

</details>

## 問題5：大きなアスキーアート

以下の出力を作ってみましょう。

```
    *
   * *

 *     *
*********
```

<details>
<summary>解答例</summary>

```ruby

puts " *     *"
```
