# 第1回 練習：putsで遊ぼう

## 今日のゴール

`puts` を使って、画面にいろいろなものを表示する。

---

## プログラムの基本構造

プログラムは基本的にこの3つでできています：

```
入力 → 処理 → 出力
```

今日やるのは「処理 → 出力」だけです。入力はまだ扱いません。まずは処理した結果を画面に出すことに集中しましょう。

---

## 準備

1. Codespacesを起動する
2. `main.rb` を開く（なければ新しく作る）
3. 書き換えるたびに、ターミナルで `ruby main.rb` を実行する

---

## 今日の目標（達成ライン）

この練習には、3つの達成ラインがあります。まずは `必須` の完了を目標にしましょう。

- `必須（全員）`：1〜3 を終える（文字表示、複数行表示、四則演算）
- `推奨（余裕がある人）`：4〜5 に進む（`#{}`、簡単なメソッド）
- `発展（早く終わった人）`：6 のチャレンジ問題に取り組む

最初から完璧を目指す必要はありません。時間内に `必須` を確実に終え、進める人は `推奨` と `発展` へ進みましょう。

## 1. 文字を表示する

```ruby
puts "こんにちは"
```

`puts` は「画面に表示しろ」という命令です。ダブルクォーテーション `"` で囲んだ文字がそのまま表示されます。

ヒント：ダブルクォーテーション `"` は、日本語キーボードでは `Shift + 2` で入力できます。

ヒント：`puts` のあとには、半角スペースを1つ入れます。

### やってみよう

自分の名前を表示してみましょう。

<details>
<summary>解答例</summary>

```ruby
puts "山田太郎"
```

</details>

---

## 2. 複数行を表示する

```ruby
puts "1行目"
puts "2行目"
puts "3行目"
```

`puts` を並べると、上から順に実行されます。プログラムは上から下に動きます。

### やってみよう

自己紹介を3行で書いてみましょう。名前、出身地、好きなもの。

<details>
<summary>解答例</summary>

```ruby
puts "山田太郎"
puts "福岡県出身"
puts "カレー・ピザ・ハンバーガーが好き"
```

</details>

---

## 3. 計算する

```ruby
puts 1 + 1
puts 10 - 3
puts 4 * 5
puts 10 / 3
```

Rubyは計算もできます。結果が画面に表示されます。

- `+` 足し算
- `-` 引き算
- `*` 掛け算
- `/` 割り算

### やってみよう

自分の年齢を計算で表示してみましょう。（例：2026 - 2006）

<details>
<summary>解答例</summary>

```ruby
puts 2026 - 2006
```

</details>

---

## 4. 文字と計算を混ぜる

```ruby
puts "1 + 1 の答えは"
puts 1 + 1
```

これだと2行に分かれます。1行にまとめたいときは：

```ruby
puts "1 + 1 の答えは #{1 + 1} です"
```

`#{}` の中に計算を書くと、結果が文字の中に埋め込まれます。

### やってみよう

「私は○歳です」を1行で表示してみましょう。○の部分は計算で。

<details>
<summary>解答例</summary>

```ruby
puts "私は #{2026 - 2006} 歳です"
```

</details>

---

## 5. Rubyに聞いてみる

`#{}` の中には計算だけでなく、Rubyへの問いかけも書けます。

```ruby
puts "こんにちは #{Time.now.year} 年"
```

`Time.now.year` は「今、何年？」とRubyに聞いています。実行すると今の西暦が表示されます。

文字を変換することもできます：

```ruby
puts "ruby".upcase
```

`upcase` は「大文字にしろ」という命令です。`RUBY` と表示されます。

### やってみよう

`"hello"` を大文字にして表示してみましょう。

<details>
<summary>解答例</summary>

```ruby
puts "hello".upcase
```

</details>

## 6. チャレンジ問題

### その1

以下の出力を `puts` だけで作ってみましょう：

```
**********
*        *
*  Ruby  *
*        *
**********
```

<details>
<summary>解答例</summary>

```ruby
puts "**********"
puts "*        *"
puts "*  Ruby  *"
puts "*        *"
puts "**********"
```

</details>

### その2

九九の7の段を表示してみましょう：

```
7 x 1 = 7
7 x 2 = 14
7 x 3 = 21
...
```

<details>
<summary>解答例</summary>

```ruby
puts "7 x 1 = #{7 * 1}"
puts "7 x 2 = #{7 * 2}"
puts "7 x 3 = #{7 * 3}"
puts "7 x 4 = #{7 * 4}"
puts "7 x 5 = #{7 * 5}"
puts "7 x 6 = #{7 * 6}"
puts "7 x 7 = #{7 * 7}"
puts "7 x 8 = #{7 * 8}"
puts "7 x 9 = #{7 * 9}"
```

</details>

### その3

以下の出力を作ってみましょう：

```
私の名前は山田太郎です
今年は2026年です
私は20歳です
10年後は30歳です
```

名前と生まれ年は自分のものに変えてください。年齢と10年後は `#{}` で計算しましょう。

<details>
<summary>解答例</summary>

```ruby
puts "私の名前は山田太郎です"
puts "今年は2026年です"
puts "私は #{2026 - 2006} 歳です"
puts "10年後は #{2026 - 2006 + 10} 歳です"
```

</details>

### その4

以下の出力を作ってみましょう：

```
     /\_/\
    ( o.o )
     > ^ <
    /|   |\
   (_|   |_)
```

ヒント：`\` を表示するには `\\` と書きます。

<details>
<summary>解答例</summary>

```ruby
puts "     /\\_/\\"
puts "    ( o.o )"
puts "     > ^ <"
puts "    /|   |\\"
puts "   (_|   |_)"
```

</details>

### その5

自由課題。`puts` と `#{}` を使って、好きなものを表示してみましょう。何でもOKです。

## まとめ

今日覚えたこと：

- `puts` で画面に表示できる
- `""` で囲むと文字になる
- `+ - * /` で計算できる
- `#{}` で文字の中に計算結果を埋め込める
- プログラムは上から下に実行される

次回は「変数」を学びます。今日 `puts` で表示したものに、名前をつけられるようになります。
