# 第1回 練習：putsで遊ぼう

## 今日のゴール

`puts` を使って、画面にいろいろなものを表示する。

---

## この練習について

コピーすれば早く終わりますが、今は自分の手で打って慣れることの方が大事です。特に最初のうちは、入力に慣れること自体が練習です。できるだけ自分で打って進めましょう。

コードを書いたら、必ず実行（例: `ruby main.rb`）してください。実行する前に「何が表示されるか」を予想してから実行すると、理解が深まります。予想と違う結果が出たら、なぜ違うのかを考えてみてください。

エラーが出ることもあります。それは失敗ではありません。エラーを読んで、原因を考えて、直して、もう一度実行する。この繰り返しが、プログラミングの力をつける一番の近道です。

各問題は、次の順番で進めてください。

1. `practice01.rb` のようなファイルを作る
2. 自分で考えてコードを書く
3. `ruby practice01.rb` で実行する
4. 動いたら、解答例を開く
5. `practice01_answer.rb` を作り、解答例を写す（コピーしてもOK）
6. `ruby practice01_answer.rb` を実行する
7. 自分の実行結果と、解答例の実行結果を比べる

実行結果が同じならOKです。
そのうえで、自分のコードと解答例を見比べて、どちらが読みやすいかも考えてみましょう。

実行結果が違う場合は、自分のコードと解答例を見比べて、どこが違うのかを確認してください。

解答例を見ることは、自分の書き方を確認し、別の考え方を学ぶための作業です。

「動いたから終わり」ではありません。
解答例と比べるところまでが、この練習です。

---

## プログラムの基本構造

プログラムは基本的にこの3つでできています：

```
入力 → 処理 → 出力
```

今日やるのは「処理 → 出力」だけです。入力はまだ扱いません。まずは処理した結果を画面に出すことに集中しましょう。

## 準備

1. Codespacesを起動する
2. `main.rb` を開く（なければ新しく作る）
3. `main.rb` に次のコードを書く

```ruby
puts "Hello, World!"
```

4. ターミナルで以下を実行する

```bash
ruby main.rb
```

`Hello, World!` と表示されたら成功です。

ここまでできたら準備完了です。
`main.rb` は動作確認用のファイルです。
ここから先の練習では、問題ごとに `practice01.rb`、`practice02.rb` のような新しいファイルを作って進めましょう。

## 📝 1. 文字を表示する ( `practice01.rb` )

```ruby
puts "こんにちは"
```

`puts` は「画面に表示しろ」という命令です。ダブルクォーテーション `"` で囲んだ文字がそのまま表示されます。

> [!TIP]
> ダブルクォーテーション `"` は、日本語キーボードでは `Shift + 2` で入力できます。

> [!TIP]
> `puts` のあとには、半角スペースを1つ入れます。

### やってみよう

自分の名前を表示してみましょう。

<details>
<summary>解答例 (practice01_answer.rb)</summary>

```ruby
# practice01_answer.rb
puts "山田太郎"
```

</details>

## 📝 2. 複数行を表示する ( `practice02.rb` )

```ruby
puts "1行目"
puts "2行目"
```

`puts` を並べると、上から順に実行されます。プログラムは上から下に動きます。


</details>

## 📝 3. 計算する ( `practice03.rb` )

```ruby
puts 1 + 1
puts 10 - 3
puts 4 * 5
puts 10 / 3
```

Rubyは計算もできます。結果が画面に表示されます。

- `+` 足し算
- `-` 引き算
- `*` 掛け算
- `/` 割り算

### やってみよう

自分の年齢を計算で表示してみましょう。（例：2026 - 2006）

<details>
<summary>解答例 (practice03_answer.rb)</summary>

```ruby
# practice03_answer.rb
puts 2026 - 2006
```

</details>

## 📝 4. 文字と計算を混ぜる ( `practice04.rb` )

```ruby
puts "1 + 1 の答えは"
puts 1 + 1
```

これだと2行に分かれます。1行にまとめたいときは：

```ruby
puts "1 + 1 の答えは #{1 + 1} です"
```

`#{}` の中に計算を書くと、結果が文字の中に埋め込まれます。

### やってみよう

「私は○歳です」を1行で表示してみましょう。○の部分は計算で。

<details>
<summary>解答例 (practice04_answer.rb)</summary>

```ruby
# practice04_answer.rb
puts "私は #{2026 - 2006} 歳です"
```

</details>

---

## 📝 5. Rubyに聞いてみる ( `practice05.rb` )

`#{}` の中には計算だけでなく、Rubyへの問いかけも書けます。

```ruby
puts "こんにちは #{Time.now.year} 年"
```

`Time.now.year` は「今、何年？」とRubyに聞いています。実行すると今の西暦が表示されます。

文字を変換することもできます：

```ruby
puts "ruby".upcase
```

`upcase` は「大文字にしろ」という命令です。`RUBY` と表示されます。

### やってみよう

`"hello"` を大文字にして表示してみましょう。

<details>
<summary>解答例 (practice05_answer.rb)</summary>

```ruby
# practice05_answer.rb
puts "hello".upcase
```

</details>

### 📝 その2 ( `practice07.rb` )
九九の7の段を表示してみましょう：

```
7 x 1 = 7
7 x 2 = 14
7 x 3 = 21
...
```

<details>
<summary>解答例 (practice07_answer.rb)</summary>

```ruby
# practice07_answer.rb
puts "7 x 1 = #{7 * 1}"
puts "7 x 2 = #{7 * 2}"
puts "7 x 3 = #{7 * 3}"
puts "7 x 4 = #{7 * 4}"
puts "7 x 5 = #{7 * 5}"
puts "7 x 6 = #{7 * 6}"
puts "7 x 7 = #{7 * 7}"
puts "7 x 8 = #{7 * 8}"
puts "7 x 9 = #{7 * 9}"
```

</details>

### 📝 その3 ( `practice08.rb` )
以下の出力を作ってみましょう：

```
私の名前は山田太郎です
今年は2026年です
私は20歳です
10年後は30歳です
```

名前と生まれ年は自分のものに変えてください。年齢と10年後は `#{}` で計算しましょう。

<details>
<summary>解答例 (practice08_answer.rb)</summary>

```ruby
# practice08_answer.rb
puts "私の名前は山田太郎です"
puts "今年は2026年です"
puts "私は #{2026 - 2006} 歳です"
puts "10年後は #{2026 - 2006 + 10} 歳です"
```

</details>

### 📝 その4 ( `practice09.rb` )
以下の出力を作ってみましょう：

```
     /\_/\
    ( o.o )
     > ^ <
    /|   |\
   (_|   |_)
```

> [!TIP]
> `\` を表示するには `\\` と書きます。

<details>

```ruby
# practice09_answer.rb
puts "     /\\_/\\"
puts "    ( o.o )"
puts "     > ^ <"
puts "    /|   |\\"
puts "   (_|   |_)"
```

</details>
